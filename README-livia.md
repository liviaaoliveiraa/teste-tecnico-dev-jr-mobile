# Perfil Dev Jr

## Descrição
Aplicativo mobile desenvolvido em React Native com Expo para apresentar o perfil profissional de uma candidata à vaga de Desenvolvedora Mobile Júnior.

## Tecnologias utilizadas
- React Native
- Expo
- JavaScript
- React Navigation
- StyleSheet
- Git
- GitHub

## Funcionalidades
- Tela de perfil profissional
- Lista de habilidades técnicas
- Exibição de projetos desenvolvidos
- Formulário de contato
- Navegação por abas inferiores

## Telas
### Perfil
Apresenta imagem, nome, cargo e descrição profissional.

### Habilidades
Lista habilidades técnicas relevantes para a vaga.

### Projetos/Contato
Mostra projetos realizados e oferece informações de contato com campo para mensagem.

## Organização do código
- App.js responsável pela navegação principal
- screens/ contém cada tela separadamente
- assets/ armazena imagem de perfil

## Diferenciais
- Layout moderno
- Interface responsiva
- Organização profissional
- Fácil manutenção
